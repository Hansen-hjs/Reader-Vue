### sass安装失败时配置（window系统）cmd 窗口首先执行命令再初始化
```
set sass_binary_site=https://npm.taobao.org/mirrors/node-sass/
```

## 初始化项目
```
npm install
```

### 运行项目
```
npm run serve
```

### 打包项目，这边建议用官方 HBuildX IDE 去打包
```
npm run build
```

### 在 HBuildX 控制台中出现版本更新提示需要手动升级一下项目
```
npm update
```
