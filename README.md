### sass安装失败时配置（window系统）cmd 窗口首先执行命令再初始化
```
set sass_binary_site=https://npm.taobao.org/mirrors/node-sass/
```

## 初始化项目
```
npm install
```

### 运行项目，APP 中连接手机调试需要借助 HbuildX
```
npm run dev
```

### 打包项目
```
npm run build
```
